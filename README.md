# Project Setup

### For development

You need a .env file containing

```
VUE_APP_API_URL="http://localhost:3000/"
```

Then run in two separate terminal

```
npm run serve
npm start
```

The client will be hosted on http://localhost:8080/

### For production

```
npm run build
npm start
```

The client will be hosted on http://localhost:3000/
